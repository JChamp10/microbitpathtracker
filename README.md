# Micro:bit Path Tracker

Turn micro:bit accelerometer CSV data into a movement path.

## Features
- Upload CSV file
- Auto-detect X, Y, Z columns
- Smooth + reduce drift
- Plot path
- Export as image

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run:
python app.py

3. Select your CSV file

Output image will be saved as:
path_output.png
